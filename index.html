Characteristics of an Algorithm

Every algorithm should satisfy these six fundamental properties. Hover over each card to learn more.

📥

Input

An algorithm accepts zero or more inputs before execution begins.
📤

Output

Every algorithm produces at least one meaningful output.
🎯

Definiteness

Every instruction must be clear, precise and unambiguous.
⏱

Finiteness

The algorithm must terminate after a finite number of steps.
⚙

Effectiveness

Every operation should be simple and executable.
🌍

Generality

It should work correctly for every valid input.

Represents the Tight Bound. Best representation of actual growth.
