# DAA Explorer 🚀

An interactive web-based learning platform for **Design and Analysis of Algorithms**.

## Features

- Algorithm characteristics visualization
- Algorithm working flow animation
- Time complexity simulation
- Space complexity visualization
- Best, Worst and Average case analysis
- Asymptotic notation comparison
- Complexity growth simulator
- Recursive algorithm visualization
- Recursion tree method
- Interactive UI with animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API
